Frequently Asked Questions

Is my data private?

Yes! All screenshots and activity data are stored locally on your computer. The only external communication is with DashScope API for AI analysis, using your own API key. You have full control over data retention and deletion.

How much does it cost?

Activity Recorder is free and open source. You only pay for the DashScope API usage, which offers a generous free tier. Typical usage costs less than $1-2 per month.

What browsers are supported?

All Chromium-based browsers: Chrome, Edge, Brave, Opera, Vivaldi, and more. Firefox and Safari support coming soon.

Can I use it offline?

Screenshots can be captured offline, but AI analysis requires internet connection to reach DashScope API. Offline activities will be analyzed when you're back online.

How accurate is the AI analysis?

Qwen-VL models are highly accurate at understanding screen content. Accuracy depends on screenshot clarity and content complexity. Success rate is typically 95%+.

Can I exclude certain applications?

Currently, the extension captures all visible screen content. Application-specific filtering is planned for a future release. You can pause recording anytime.

How do I get a DashScope API key?

Sign up for a free account at dashscope.aliyun.com. Navigate to the API Keys section and create a new key. Free tier includes 1 million tokens per month.
